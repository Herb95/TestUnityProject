# vscode 安装GitLess 简单命令

``` 基础命令
    gl init - 创建一个空仓库或从现有的远程仓库创建一个
    gl status - 显示回购的状态
    gl track - 开始跟踪文件的更改
    gl untrack - 停止跟踪文件的更改
    gl diff - 显示文件更改
    gl commit - 记录本地仓库的变化
    gl checkout - checkout已提交的文件版本
    gl history - 显示提交历史记录
    gl branch - 列出，创建，编辑或删除分支
    gl switch - 切换分支
    gl tag - 列出，创建或删除标签
    gl merge - 将一个分支的不同变化合并到另一个分支上
    gl fuse - 将一个分支的不同变化融合到另一个分支上
    gl resolve - 将冲突标记为已解决的文件
    gl publish - 发布上游提交
    gl remote - 列出，创建，编辑或删除遥控器
```
